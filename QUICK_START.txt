## 🚀 How to Use

### Option 1: Open Directly in Browser
1. Navigate to: `c:\Users\User\Documents\project-website\`
2. Double-click `index.html`
3. The website opens in your default browser

### Option 2: Use a Local Server (Recommended)

**Windows - Python:**
```bash
cd c:\Users\User\Documents\project-website
python -m http.server 8000
# Then open http://localhost:8000 in browser
```

**Windows - Using VS Code:**
1. Install "Live Server" extension
2. Right-click on `index.html`
3. Select "Open with Live Server"

**Any OS - Using Node.js:**
```bash
cd project-website
npx http-server
# Follow the URL provided
```

## 📝 Customization Guide

### 1. Change Project Title
- Open any HTML file and search: "AI-Powered Student Project Management System"
- Replace with your project name
- Do this in all HTML files for consistency

### 2. Update Team Members
- Open `about-us.html` and `contact-us.html`
- Update names, roles, emails, and phone numbers
- Replace emoji icons if desired

### 3. Change Colors
- Open `css/style.css`
- Find the `:root` section (top of file)
- Modify these variables:
  ```css
  --primary-color: #2c3e50;      /* Dark blue */
  --secondary-color: #3498db;    /* Light blue */
  --accent-color: #e74c3c;       /* Red */
  --light-bg: #ecf0f1;           /* Light gray */
  ```

### 4. Add Content
- **Documents**: Place PDFs in `docs/` folder
- **Presentations**: Place files in `slides/` folder
- **Images**: Place images in `images/` folder
- Update file references in HTML

### 5. Update Contact Information
- Change email: `project@university.edu`
- Change phone: `+1 (234) 567-890`
- Change location details
- Update links to team members

## 🆘 Need Help?

### Common Tasks
- **Change email**: Search for "project@university.edu" in all files
- **Change phone**: Search for "+1 (234) 567-890" in all files
- **Add navigation item**: Add `<li><a href="newpage.html">New Page</a></li>` in nav-menu
- **Remove a page**: Delete the HTML file and remove its nav link

### If Something Breaks
1. Check browser console (F12 → Console tab) for errors
2. Verify all file paths match exactly (case-sensitive)
3. Clear browser cache (Ctrl+F5)
4. Check that style.css and script.js are in correct folders
