AI-Powered Personal Fashion Assistant

StyleMate helps users choose the perfect outfits and accessories using AI-based skin tone and body shape analysis.

Explore Project

Project Overview

🎯 Literature survey

Existing fashion systems use AI and machine learning to suggest outfits, but they mainly focus on clothing items. Most systems do not consider important factors like body shape, skin tone, or event type. E-commerce platforms rely on user history, which leads to less accurate recommendations. Therefore, a more personalized and integrated solution is needed.

💡 Methodology

The system uses AI and computer vision to analyze user images and detect skin tone and body shape. Based on this, it provides personalized outfit and accessories recommendations. It also considers event type and manages a digital wardrobe system.

⚙️ Technologies Used

Frontend: React.js for building the user interface

Backend: FastAPI for handling APIs and system logic

Machine Learning: PyTorch for model development

Model: MobileNetV2 for image analysis (skin tone & body shape)

Integration: REST API for communication between frontend and backend

❗ Research Problem

Users face difficulty selecting outfits that match their body shape, skin tone, and event requirements, leading to poor styling decisions, time wastage, and increased product return rates in online fashion retail.

🔍 Research Gap

Existing fashion recommendation systems provide only basic suggestions, rarely consider body shape and skin tone, lack event-based styling support, do not integrate accessory recommendations, and fail to include smart wardrobe management features.

🎯 Objectives

To develop StyleMate, an AI-powered fashion assistant that delivers personalized outfit and accessory recommendations using machine learning and computer vision, while supporting event-based styling and smart wardrobe management.

Key Features

🎨 Skin Tone Detection

Detects user skin tone using image processing to suggest suitable colors that enhance natural beauty.

👤 Body Shape Analysis

Analyzes body shape and proportions to recommend clothing styles that flatter and suit individual physiques.

👗 Outfit Recommendations

Generates personalized outfit combinations based on user preferences, trends, and wardrobe inventory.

✨ Accessories Suggestion

Recommends complementary accessories for complete looks.

📅 Event-Based Styling

Provides outfit suggestions tailored to specific occasions and events you're planning to attend.

🗂️ Wardrobe Management

Helps organize and manage your digital wardrobe with smart categorization and inventory tracking.
